# 지원고고 (ApplyGogo)

## 📖 문서 (Documentation)

모든 프로젝트 문서가 **[docs/PRD.md](./docs/PRD.md)** 하나로 통합되었습니다.
이곳에서 다음 내용을 모두 확인할 수 있습니다:

- 제품 개요 및 기획 의도
- **구현 및 디자인 지침 (UI Preservation)**
- 유저 스토리 및 워크플로우
- 기술 아키텍처 (DB 스키마)
- 디자인 시스템 가이드

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
```
